Завдання 1.1

Умова

Написати функції createProduct(obj, callback), logProduct(product), logTotalPrice(product). Функція createProduct повинна додавати унікальний id до об'єкта товару та передавати його у callback-функцію.

Код

function createProduct(obj, callback) {

    const product = {
        ...obj,
        id: Date.now()
    };

    callback(product);
}

function logProduct(product) {
    console.log(product);
}

function logTotalPrice(product) {
    console.log(product.price * product.quantity);
}

Завдання 1.10

Умова

Написати функцію checkBrackets(str), яка перевіряє правильність закриття дужок (), {} та []. Якщо рядок містить коректний код, функція повертає true, інакше false.

Код

function checkBrackets(str) {

    const stack = [];

    const brackets = {
        ")": "(",
        "}": "{",
        "]": "["
    };

    for (let char of str) {

        if (
            char === "(" ||
            char === "{" ||
            char === "["
        ) {
            stack.push(char);
        }

        if (
            char === ")" ||
            char === "}" ||
            char === "]"
        ) {

            if (
                stack.pop() !== brackets[char]
            ) {
                return false;
            }
        }
    }

    return stack.length === 0;
}

Завдання 2.7

Умова

Створити клас Calculator з підтримкою method chaining для виконання математичних операцій.

Код

class Calculator {

    constructor(value = 0) {
        this.value = value;
    }

    add(num) {
        this.value += num;
        return this;
    }

    subtract(num) {
        this.value -= num;
        return this;
    }

    multiply(num) {
        this.value *= num;
        return this;
    }

    getResult() {
        return this.value;
    }
}
