Мета

Набути практичних навичок використання CSS, підключення зовнішнього файлу стилів, застосування різних типів селекторів (тег, клас, id, інші), а також оформлення тексту, таблиць і списків.

Файли

1. Селектори тегу (tag selectors)

Селектор тегу застосовується до усіх елементів певного HTML-тега. Наприклад: body, h1, p, a.

Фрагмент CSS

/* Селектор тегу */
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
}

a:hover {
  text-decoration: underline;
}

Демонстрація

Цей текст — тег <p>, стилізується правилами для тегів.

2. Селектори класу (class selectors)

Селектор класу визначається через .class і може застосовуватись до багатьох елементів.

Фрагмент HTML

<p class="highlight">
  Приклад тексту зі стилем класу
</p>

Фрагмент CSS

/* Селектор класу */
.highlight {
  background: #fff4a3;
  font-weight: 700;
}

Цей текст оформлено стилем класу .highlight.

3. Селектори ідентифікатора (id selectors)

Ідентифікатор — унікальний. Стилі задаються через #id.

Фрагмент HTML

<div id="selector-id-demo">...</div>

Фрагмент CSS

/* Селектор id */
#selector-id-demo {
  border-color: #2f8f2f;
  background: #effff0;
}

ID-селектор

Цей блок стилізовано через #selector-id-demo.

4. Інші селектори

Для демонстрації використано:

  • Універсальний — *
  • Вкладений — .demo-box p
  • Дочірній — .demo-box > h4
  • Сусідній — h4 + p
  • Атрибутний — a[href^="mailto:"]

Фрагмент CSS

/* Універсальний */
* { box-sizing: border-box; }

/* Вкладений */
.demo-box p { margin: 0; }

/* Дочірній */
.demo-box > h4 { text-transform: uppercase; }

/* Сусідній */
h4 + p { margin-top: 4px; }

/* Атрибутний */
a[href^="mailto:"] { font-weight: 700; }

Демонстрація інших селекторів

Цей текст оформлено вкладеним селектором .demo-box p.

Висновки

У ході виконання лабораторної роботи №2 я навчилась підключати зовнішні таблиці стилів, використовувати CSS-селектори (тегу, класу, id), а також інші селектори (універсальний, вкладений, дочірній, сусідній, атрибутний). Стилі застосовано для тексту, таблиці та списків.

Контрольні питання

1. Для чого використовуються каскадні таблиці стилів?

CSS відокремлює оформлення від структури HTML-документа та дозволяє керувати зовнішнім виглядом елементів.

2. Які методи визначення стилів у HTML-документі ви знаєте?

Inline-стилі (у атрибуті style), внутрішні стилі (всередині <style>), зовнішній файл стилів (через <link>).

3. У чому різниця між каскадуванням і спадкуванням?

Каскадування визначає пріоритет стилів при конфліктах, спадкування — передавання частини стилів від батьківських елементів до дочірніх.

4. Як оформлюються селектори тегу і класу?

Селектор тегу — назва тега (p { }), селектор класу — через крапку (.item { }).

5. Для чого використовується ідентифікатор?

Для унікальної ідентифікації елемента, створення якорів у навігації, та стилізації через #id.
