Структура WEB-застосунку

HTML-документ складається з таких основних частин:

  • <header> — логотип і навігаційне меню
  • <main> — головний контент сторінки
  • <section> — окремі блоки з товарами, інформацією, контактами
  • <footer> — підвал із джерелами

HTML-код зображення


    <img rc="images/logo.png" width="120" height="100">

Тег <img> використовується для вставки зображень. Основні атрибути - src (шлях до файлу) і width, hight - розміри картинки.

Контрольні питання

1. Яка структура HTML-документа?

Структура HTML-документа складається з оголошення <!DOCTYPE html>, кореневого елемента <html>, блоку <head> з метаданими та блоку <body>, у якому розміщується видимий вміст веб-сторінки. Усі елементи документа організовані у вигляді вкладеної структури.

2. Дати визначення тега.

Тег — це елемент HTML-мови, який позначає структуру або функцію частини документа. Більшість тегів мають відкривальний і закривальний варіанти та можуть містити атрибути, які уточнюють їх поведінку або зовнішній вигляд.

3. Які теги використовуються при роботі з таблицями?

Для створення таблиць використовується тег <table>. Рядки таблиці визначаються тегом <tr>, заголовкові комірки — <th>, звичайні комірки — <td>. Також можуть використовуватися елементи <thead>, <tbody> і <tfoot> для структурування таблиці.

4. Які теги використовуються при роботі із зображеннями?

Для вставки зображень використовується тег <img>, який є порожнім і не має закривального тега. Основні атрибути: src — шлях до файлу, alt — текстовий опис, width і height — розміри зображення.

5. Які теги використовуються при роботі зі списками?

Для маркованих списків використовується тег <ul>, для нумерованих — <ol>. Окремі елементи списку позначаються тегом <li>. Усередині пунктів можуть знаходитися інші теги або вкладені списки.

6. Перерахувати засоби опису таблиць в HTML.

Для опису таблиць використовуються такі засоби: <caption> — заголовок таблиці, border — межі комірок, cellpadding — внутрішні відступи комірок, cellspacing — відстань між комірками, colspan і rowspan — об'єднання комірок. Також використовуються структурні теги <thead>, <tbody> та <tfoot>.
